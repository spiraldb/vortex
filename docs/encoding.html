class vortex.encoding.Array#

The root class for all Vortex arrays.

__len__()#

Return len(self).

dtype#

The data type of this array.

Returns#

vortex.dtype.DType

Examples#

By default, vortex.encoding.encode() uses the largest available bit-width:

>>> vortex.encode([1, 2, 3]).dtype
int(64, False)

Including a None forces a nullable type:

>>> vortex.encode([1, None, 2, 3]).dtype
int(64, True)

A UTF-8 string array:

>>> vortex.encode(['hello, ', 'is', 'it', 'me?']).dtype
utf8(False)
take(indices)#

Filter, permute, and/or repeat elements by their index.

Returns#

vortex.encoding.Array

Examples#

Keep only the first and third elements:

>>> a = vortex.encode(['a', 'b', 'c', 'd'])
>>> indices = vortex.encode([0, 2])
>>> a.take(indices).to_arrow()
<pyarrow.lib.StringArray object at ...>
[
  "a",
  "c"
]

Permute and repeat the first and second elements:

>>> a = vortex.encode(['a', 'b', 'c', 'd'])
>>> indices = vortex.encode([0, 1, 1, 0])
>>> a.take(indices).to_arrow()
<pyarrow.lib.StringArray object at ...>
[
  "a",
  "b",
  "b",
  "a"
]

to_pandas(*, name: str | None = None, flatten: bool = False)#

Construct a Pandas dataframe from this Vortex array.

Parameters#

objpyarrow.Array or list

The elements of this array or list become the elements of the Vortex array.

namestr, optional

The name of the column in the newly created dataframe. If unspecified, use x.

flattenbool

If True, Struct columns are flattened in the dataframe. See the examples.

Returns#

pandas.DataFrame

Examples#

Construct a pandas.DataFrame with one column named animals from the contents of a Vortex array:

>>> array = vortex.encode(['dog', 'cat', 'mouse', 'rat'])
>>> array.to_pandas(name='animals')
  animals
0     dog
1     cat
2   mouse
3     rat

Construct a pandas.DataFrame with the default column name:

>>> array = vortex.encode(['dog', 'cat', 'mouse', 'rat'])
>>> array.to_pandas()
       x
0    dog
1    cat
2  mouse
3    rat

Construct a dataframe with a Struct-typed column:

>>> array = vortex.encode([
...     {'name': 'Joseph', 'age': 25},
...     {'name': 'Narendra', 'age': 31},
...     {'name': 'Angela', 'age': 33},
...     {'name': 'Mikhail', 'age': 57},
... ])
>>> array.to_pandas()
                                 x
0    {'age': 25, 'name': 'Joseph'}
1  {'age': 31, 'name': 'Narendra'}
2    {'age': 33, 'name': 'Angela'}
3   {'age': 57, 'name': 'Mikhail'}

Lift the struct fields to the top-level in the dataframe:

>>> array.to_pandas(flatten=True)
   x.age    x.name
0     25    Joseph
1     31  Narendra
2     33    Angela
3     57   Mikhail
vortex.encoding.compress(array)#

Attempt to compress a vortex array.

Parameters#

array : vortex.encoding.Array

The array.

Examples#

Compress a very sparse array of integers:

>>> a = vortex.encode([42 for _ in range(1000)])
>>> str(vortex.encoding.compress(a))
'vortex.constant(0x0a)(i64, len=1000)'

Compress an array of increasing integers:

>>> a = vortex.encode(list(range(1000)))
>>> str(vortex.encoding.compress(a))
'fastlanes.for(0x0f)(i64, len=1000)'

Compress an array of increasing floating-point numbers and a few nulls:

>>> a = vortex.encode([
...     float(x) if x % 20 != 0 else None
...     for x in range(1000)
... ])
>>> str(vortex.encoding.compress(a))
'vortex.alp(0x0d)(f64?, len=1000)'
vortex.encoding.encode(obj: Array | list) → Array#

The main entry point for creating enc arrays from other Python objects.

Parameters#

objpyarrow.Array or list

The elements of this array or list become the elements of the Vortex array.

Returns#

vortex.encoding.Array

Examples#

A Vortex array containing the first three integers.

>>> vortex.encode([1, 2, 3]).to_arrow()
<pyarrow.lib.Int64Array object at ...>
[
  1,
  2,
  3
]

The same Vortex array with a null value in the third position.

>>> vortex.encode([1, 2, None, 3]).to_arrow()
<pyarrow.lib.Int64Array object at ...>
[
  1,
  2,
  null,
  3
]

Initialize a Vortex array from an Arrow array:

>>> arrow = pyarrow.array(['Hello', 'it', 'is', 'me'])
>>> vortex.encode(arrow).to_arrow()
<pyarrow.lib.StringArray object at ...>
[
  "Hello",
  "it",
  "is",
  "me"
]
