Enum vortex_scalar::ScalarValue

source · 
pub enum ScalarValue {
    Bool(bool),
    Primitive(PValue),
    Buffer(Buffer),
    BufferString(BufferString),
    List(Arc<[ScalarValue]>),
    Null,
}
Expand description

Represents the internal data of a scalar value. Must be interpreted by wrapping up with a DType to make a Scalar.

Note that these values can be deserialized from JSON or other formats. So a PValue may not have the correct width for what the DType expects. This means primitive values must be cast on-read.
